# React + TypeScript + Vite

## Development

```bash
pnpm dev
```

## Build and run

```bash
pnpm build
```

```bash
pnpm preview
```

## Test

```bash
pnpm test:ui
```

## Docker

Build the docker image with:

```bash
docker build -t trumpet .
```

Run the docker image with:

```bash
docker run -p 3000:3000 trumpet
```
